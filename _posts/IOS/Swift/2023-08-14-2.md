🍏알고리즘 문제를 풀며 정리한 기초 알고리즘 & 문법입니다.
{: .notice--warning}

> 문자열의 길이를 알고싶을 때

```swift
let myName : String = "Johnson"
print(myName.count)

// 7
```

> 배열의 길이를 알고싶을 때

```swift
let Array : Array<Int> = [1,2,3,4,5]
print(Array.count)

// 5
```
